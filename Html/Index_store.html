About us

MetalMC is a Minecraft 1.13-1.19 PvP server where you can
practice every common gamemode in the current PvP version of Minecraft.
The server offers many different gamemodes you can duel in, 7 different FFA’s
you can play and tons of new events being hosted every week.
The server is located in two different regions, both North America
and Europe so as many players can enjoy their time on MetalMC.
The server was originally founded by Coldmonke but in the way
of processing the server he couldn’t do it alone so he promoted
Criticaking to owner as Criticalking would be the main developer
in the server doing mostly everything related to developping the server.
Coldmonke being the main builder and
Criticalking being the main developer those two are now the owners of MetalMC.
This website was made by RLRGC and Criticalking.
